## Overview

SocialTimeline displays a timeline of social events. You must provide a `messages` array with this shape:

```js
{
  actor: {
    name: 'Name',
    avatar: 'Link to avatar',
  },
  verb: 'sent you a message to',
  object: {
    id: 'object-id',
    definition: {
      name: 'Group 3',
    }
  }
  timestamp: '2022-07-05T18:30:32.360Z'
}
```

By default SocialTimelines uses the navigator locale but you can provide the locale you want
