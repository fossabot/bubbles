## ColorPicker variants

ColorPicker can be compact to hide the color swatches and make it smaller:

<ThemeProvider>
  <Group>
    <ColorPicker />
    <Space></Space>
    <ColorPicker compact />
  </Group>
</ThemeProvider>

It can also has a mode for only use the HSL model:

<ThemeProvider>
  <Group>
    <ColorPicker useHsl />
    <Space></Space>
    <ColorPicker useHsl compact />
  </Group>
</ThemeProvider>

And it can also be set to fullwidth:

<ThemeProvider>
  <Box style={{ width: '50%' }}>
    <ColorPicker fullWidth />
  </Box>
  <Box style={{ width: '50%' }}>
    <ColorPicker useHsl fullWidth />
  </Box>
</ThemeProvider>
